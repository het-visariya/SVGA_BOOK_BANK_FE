# 📖 Project Overview

SVGA Book Bank Frontend is the client-side application of the Book Bank Management System developed during my internship.

The platform digitizes the traditional community book lending process by allowing students to browse available books, request library resources, submit special procurement requests, and track the complete approval workflow online.

The application focuses on providing a clean, responsive, and user-friendly experience for students while seamlessly integrating with the backend APIs.

---

# 🎯 Purpose

Traditional book bank systems are often paper-based, difficult to manage, and lack transparency.

This project aims to:

- Digitize the complete book request workflow
- Reduce manual paperwork
- Improve accessibility for students
- Provide real-time request tracking
- Simplify communication between students and administrators
- Build a scalable frontend for future enhancements

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/het-visariya/SVGA_BOOK_BANK_FE.git
```

Move into project

```bash
cd SVGA_BOOK_BANK_FE
```

Install dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```

or

```bash
npm start
```

Open

```
http://localhost:5173
```
